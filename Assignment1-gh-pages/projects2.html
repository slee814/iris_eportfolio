Artifact title: Picture Sharing
Description: A simple web application that lets users share with others images with an option to include a title.
Reflection:
This artifact is one of my best work because although it is simple, it demonstrate some of the fundamentals of developing web applications. For example, this web application is able to extract user’s input information and display it on the website.

One challenge that I had when I develop the application was to display user’s post in order. Initially, user’s posts were not displayed in the order that they were posted. I problem solved it by backtracking what I did and then break the problem into smaller pieces. Eventually I was able to determine what I did wrong and correct it. This is an effective problem solving strategy that I have used to apply on other situations.
